# Vault Folder MOC (Map of Content)

This is a high-level outline of all folders and their current/planned use in the Second Braincell Vault system.

---

## Root Level
- `HUD.md`: Displays the active file tree and structure in every view

---

## 🎭 Narrators/
- Core narrator definitions and triggers
- Each `.md` file defines one narrator’s logic and behavior
- Future: possible narrator packs with bundled personalities

---

## 🩺 Health/
- Symptom logs, cueing patterns, and hypermobility tracking
- Watchlist for connective tissue concerns, RLS, GI symptoms

---

## ⚙️ System/
- Daily Log Template and HUD rules
- Discomfort escalation logic
- Travel overrides (future)

---

## 📅 Events/
- Countdown logic and active to-do system
- Trip planning and checklists (expandable to `Travel_Mode.md`)

---

## 📜 Logs/
- System_Changelog.md (version history)
- Vault_Worktime_Tracker.md (meta effort log)

## 📓 Meta/
