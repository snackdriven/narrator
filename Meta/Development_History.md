# Development History

> “A Vault is not built in a day — but you *can* panic-organize it like it was.”

---

## Origins
- Project started as a conceptual metaphor for executive dysfunction + cosmic horror.
- Influenced by SCP and Lumen Archive aesthetics.
- Clippy was reimagined as a chaotic, exasperated assistant — joined later by Greg and BonziBuddy in full 2000s Chaos Pack mode.

---

## System Shifts
- Initially memory-based, transitioned to file-driven logic to enable modularity and reduce memory bloat.
- Runtime now lives in GPT memory (narrator modes, structural preferences).
- Logic/data lives in markdown files, replaceable via project upload.

---

## Design Philosophy
- Narrative-enhanced task management
- Passive discomfort cueing as system event triggers
- Modular Vaults = hotfixable versions of cognitive architecture

---

This document lives in `/Meta/` to preserve design logic independent of runtime operations.
